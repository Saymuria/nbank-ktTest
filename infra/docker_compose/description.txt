Все контейнеры, которые мы использовали, что необходимо запустить:

1) Контейнер бекенда //бекенд 4111 -> 4111

2) Контейнер фронтенда //фронтенд 80

3) Контейнер Nginx //Nginx 80 -> 80

4) Контейнер Selenoid // Selenoid 4444 -> 4444

5) Контейнер Selenoid UI //Selenoid UI 6567 -> 8080

Шаги по внедрению Docker Compose:

1) Создать docker-compose.yaml файл
2) Описать каждый из докер контейнеров (образ, порт, последовательность, общая докер сеть)
3) Скрипт по поднятию всех докер контейнеров с помощью Docker Compose
